fromscratch was built so I could practice web development & learn new technologies. I intend to update this app with a robust set of features (shown below) to help New Zealander's be more informed in their dinner choices.

If you'd like to ask me any questions or submit any bugs, message me!

Features planned & in progress:

  • Switch between measuring types (cups, tsp,mls)
  • All custom recipe layouts
  • A 1-2-3 step of choosing supermarket, directions to said recipe, cooking directions
  • Substitutions of ingredients based on price, availability or allergies
